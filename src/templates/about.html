Librerias utilizadas

Bulma

Bulma es un framework de CSS de codigo libre basado en Flexbox, que es un modelo unidimensional de layout, que distribuye el espacio entre los ítems de una interfaz y mejorar las capacidades de alineación


Flask

Flask es un framework minimalista escrito en Python que permite crear aplicaciones web rápidamente y con un mínimo número de líneas de código


Matplotlib

Matplotlib es librería de gráficos python en 2D que produce cifras de calidad de publicación en una variedad de formatos de papel y entornos interactivos en todas las plataformas


Graphviz

Graphviz es un conjunto de herramientas de software para el diseño de diagramas definido en el lenguaje descriptivo DOT.1 Fue desarrollado por AT&T Labs2 y liberado como software libre con licencie tipo Eclipse.


Pydot

Pydot es un modulo con una interfaz completa de Graphviz para crear, manejar, modificar y procesar grafos en lenguaje DOT.
